# [图标](/icon)

用法 ：  
策略组后添加 img-url=https://raw.githubusercontent.com/Gavinuuup/QuantumultX/master/icon/hk.png  
举例 ：  
static=香港, 🇭🇰01, img-url=https://raw.githubusercontent.com/Gavinuuup/QuantumultX/master/icon/hk.png  
这样就调用了香港图标
